# Raporlar (Releases)

YKS "sallamasyon" araştırma serisinin yayımlanmış PDF raporları.
Yazarlar: Hamza Yiğit Kültür, İsmail Güler.

| Dosya | İçerik |
|-------|--------|
| `Rapor1_Model_Zoo_ve_Sik_Tahmini.pdf` | Veri çıkarımı, keşifsel analiz ve sıralı tahmin model zoo'su (baseline'lar + MLP/CNN/GRU/Transformer). 2018–2024 eğitim, 2025 değerlendirme. |
| `Rapor2_Sik_Dengeleme_ve_Imputation.pdf` | Şık dengeleme sinyali ve sızıntısız imputation (boşluk doldurma) değerlendirmesi. Dengeleme/Markov/Hybrid/SmartHybrid/Unified + GRU/MaskedBERT. |
| `Rapor3_Sifirdan_Tahmin.pdf` | Hiçbir cevap bilinmeden (k=0) tüm cevap anahtarının kestirimi. PermCycle ve diğer cold-start modelleri. |
| `Rapor4_Cevap_Anahtari_Atlasi.pdf` | Model içermeyen, salt betimsel atlas: 2018–2025 sekiz yılın şık dağılımı, ısı haritaları, PCA embedding ve yapı analizi. |

Tüm raporlar `reports/report*.tex` kaynaklarından `xelatex` ile derlenmiştir.
Zaman damgalı arşiv kopyaları `reports/archive/` altındadır.
